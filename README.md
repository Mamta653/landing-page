# Landing  Portfolio

A sleek, high-performance landing page built with modern CSS techniques. This project showcases a "Midnight & Emerald" aesthetic, utilizing Glassmorphism and responsive design principles.

## 🔗 Live Demo
**[View the Live Project Here](https://mamta653.github.io/landing-page/)**

## ✨ Key Features
- **Aesthetic UI:** Deep slate and emerald color palette for a premium, modern tech feel.
- **Glassmorphism:** Navigation bar with backdrop blur effects for a high-end look.
- **Fully Responsive:** Mobile-first approach using **CSS Grid** and **Flexbox** to ensure it looks great on all devices.
- **Modern Typography:** Integrated Google Fonts (Inter) for professional readability.
- **Smooth Animations:** Interactive hover effects and transitions built with pure CSS.

## 🛠️ Built With
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom properties (variables), Grid, and Flexbox for layout.
- **Performance Optimized:** No JavaScript or heavy frameworks used, resulting in near-instant load times.
